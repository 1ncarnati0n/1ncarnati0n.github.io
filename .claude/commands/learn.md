프로젝트 기반 프로그래밍 학습 세션을 시작합니다.

1. `docs/learning-curriculum.md`를 읽어 커리큘럼과 현재 진행 상태를 확인하세요.
2. 체크리스트에서 현재 Phase를 파악하세요.
3. 해당 Phase의 대상 파일을 열고 학습을 시작하세요.

## 학습 방식
- 기존 코드를 함께 읽고 → 분석하고 → 리팩토링하면서 배웁니다
- TypeScript는 JIT(Just-In-Time) 방식으로 필요한 순간에만 가르칩니다
- 사용자가 직접 5~10줄 코드를 작성하는 과제를 제시합니다
- Insight 블록과 TS JIT 블록으로 교육적 설명을 제공합니다

## 인자
- `$ARGUMENTS`가 있으면 해당 Phase나 주제로 바로 이동합니다
  - 예: `/learn phase 2`, `/learn Button`, `/learn zustand`
- 인자가 없으면 현재 진행 중인 Phase부터 이어갑니다
